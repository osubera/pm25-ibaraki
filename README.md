# pm25-ibaraki
==============

Visualizes PM 2.5 concentration shown at Ibaraki prefecture agency web site. 
Works with R. 

## Data
-------
- [Air pollution monitoring service, Ibaraki prefecture, Japan](http://www.taiki.pref.ibaraki.jp/data.asp)
